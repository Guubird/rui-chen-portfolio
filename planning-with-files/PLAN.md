# Implementation Plan

- Create README first.
- Create blueprint, prompt, decisions, component plan, design system, and TODO checklist.
- Inspect WorkTrace, ClipDock, and evidence assets.
- Copy useful local assets into the project.

## Phase 2: Project Setup

- Add Vite + React + TypeScript app files.
- Configure Tailwind CSS.
- Install Framer Motion, Three.js, and lucide-react.
- Add global CSS, theme tokens, and reduced-motion handling.

## Phase 3: Core Experience

- Build the hero with large typography and abstract Three.js crystal.
- Add pointer-responsive crystal movement.
- Add mouse ripple/particle effect layer.
- Build header and section navigation.

## Phase 4: Content Sections

- Build Selected Projects.
- Build Research / Awards / Certificates.
- Build Technical Stack.
- Build About.
- Build Contact.
- Build Footer.

## Phase 5: Responsive Polish

- Check desktop, tablet, and mobile layouts.
- Make sure text does not overlap.
- Make evidence images readable but secondary.
- Tune spacing and visual hierarchy.

## Phase 6: Verification

- Run `npm install`.
- Run `npm run build`.
- Start dev server if possible.
- Smoke test the local site in browser.
- Update README and DEVELOPMENT_LOG.
